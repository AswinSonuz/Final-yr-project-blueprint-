Project Overview

Problem
Structural Safety
Manual inspection of concrete surfaces is slow, inconsistent, and expensive. Cracks signal structural degradation and must be caught early.
Solution
Deep Learning Vision
A CNN-based image classification + segmentation pipeline that detects crack type, location, and severity from camera photos in real time.
Output
Web Dashboard
Upload image → get crack type, confidence score, annotated heatmap, and severity report. Works on mobile camera too.
Novelty
Classification + Severity
Not just detection — classify crack as hairline, structural, fatigue, or corrosion-induced and give a 1–5 severity score.

03 —

7-Month Timeline

MONTH 01 Research & Setup FOUNDATION

Goals

  • Literature review: read 5–8 papers on crack detection (SDNET, CrackForest, DeepCrack)
  • Understand CNN basics: conv layers, pooling, batch norm, ReLU, softmax
  • Study ResNet-50 and EfficientNet-B4 architectures
  • Setup Python environment: PyTorch, OpenCV, FastAPI, Docker
  • Download and explore SDNET2018 dataset — understand its structure
  • Write Problem Statement, Objectives, and Scope in project report
  • Define crack classes: No Crack / Hairline / Structural / Wide / Spalling
✓ DELIVERABLE: Literature Review + Project Report Chapter 1
MONTH 02 Data Collection & Preprocessing DATA

Goals

  • Finalize dataset: SDNET2018 + CrackForest + optionally photograph walls on campus
  • Label images with RoboFlow or LabelImg for segmentation masks
  • Build preprocessing pipeline: resize to 224×224, normalize, convert to tensor
  • Apply augmentations: flip, rotate, brightness, Gaussian noise, CLAHE
  • Split dataset: 70% train / 15% val / 15% test — ensure class balance
  • Visualize class distribution and sample images with OpenCV
  • Handle imbalanced classes with weighted sampling or oversampling
✓ DELIVERABLE: Cleaned Dataset + Data Pipeline Script + Report Chapter 2
MONTH 03 Model Development — Classification MODEL v1

Goals

  • Build custom CNN baseline (3–5 conv blocks) for binary classification
  • Implement transfer learning with ResNet-50 (freeze backbone, train head)
  • Try EfficientNet-B4 as alternative backbone
  • Track training: loss curves, accuracy, confusion matrix with MLflow
  • Implement early stopping, LR scheduler, dropout regularization
  • Achieve ≥90% accuracy on validation set before moving on
  • Grad-CAM visualizations to verify model looks at cracks, not background
✓ DELIVERABLE: Trained Classification Model (.pt file) + Training Notebook
MONTH 04 Segmentation + Severity Scoring MODEL v2

Goals

  • Implement U-Net for pixel-level crack mask prediction
  • Train segmentation model on labeled mask dataset
  • Compute crack width and length from predicted mask (OpenCV contour analysis)
  • Design severity scoring: ratio of crack pixels to image size → score 1–5
  • Combine classification + segmentation into unified inference pipeline
  • Evaluate with IoU (Intersection over Union) and Dice coefficient
  • Test on unseen images — qualitative and quantitative review
✓ DELIVERABLE: Segmentation Model + Severity Module + Combined Pipeline
